  • 硬件

    • 开发板

    • 信号发生器

  • 软件

    • PC 端的串口终端软件，用于 PC 和开发板进行串口通信

    • Luban-Lite 自带的 test_cap 命令行工具

软件配置

  1. 在 Luban-Lite 根目录下执行 scons --menuconfig，进入 menuconfig 的功能配置界面，按如下选择打开 test_cap 命令行工具：
    Drivers options --->
    Drivers examples --->
        [*] Enable CAP driver test command

  2. 同时，按照如下选择，打开中断打印配置：

    Rt-Thread options --->
    RT-Thread Components --->
                    Utilities --->
        [*] Enable ISR log.

测试 CAP

信号发生器配置输出 3.3 V、 适当周期和占空比的方波，接入 PC6 （以 CAP0 输入引脚为例）。

在命令行中执行 test_cap0 进行测试。
test_cap 0
