userid 命令参考指南

25 Dec 2024
Read time: 1 minute(s)

U-Boot 测试命令

U-Boot 中通过命令行提供了 UserID 的增删改查等操作，具体命令的使用可通过 help 命令查看：

  • userid help

  • userid init ：从 UserID 分区加载和初始化 UserID 数据：

  • userid list ：查看当前的 ID 列表

  • userid writehex name offset data：写入新 ID

    示例：
    userid writestr sn 0 abcdefgh1234567
  • userid dump name：查看 ID 值
    示例：
    userid dump sn

  • userid lock ：锁定 UserID

  • userid unlock ：解锁 UserID

Linux 测试命令

Linux 中，如果编译时选中了 userid library 包，则默认提供 userid 命令。

  • userid list <partition>：查看当前 ID 列表

    示例：
    userid list /dev/mtd2

  • userid dump <partition> id：查看 ID 值

    示例：
    userid dump /dev/mtd2 sn
