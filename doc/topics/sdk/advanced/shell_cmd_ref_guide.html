Shell 命令行工具参考指南

19 Dec 2024

RT-Thread Shell 命令行是 RT-Thread 操作系统中的一个重要功能，允许用户通过命令行与系统进行交互，执行各种操作和调试任务。在 RT-Thread Shell 中，用户可以输入各种命令来查看系统状态、管理线程和设备、控制 GPIO 等。

date

查看时区配置。
date
输出示例如下：
Tue Jan  601:41:27 UTC 1970

echo

echo 9 > brightness

free

显示系统中的内存使用情况。
