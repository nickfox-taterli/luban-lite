make pkg-

27 Dec 2024
Read time: 1 minute(s)

make pkg

完成从 extract/ patch/ ../ build/ install 的全部编译流程。

示例：
make qt

make pkg-extract

仅对源码包进行解压。

  • 示例 1：解压缩 ethtool 包
    make ethtool-extract
  • 示例 2：把 dl 中的压缩包解压到 source 目录
    make qt-extract

make pkg-patch

仅对源码包进行 patch，如有。

示例：把 package 中的 patch 打到 source 源码中：
make qt-patch

make pkg-reconfigure

对该源码包重新执行配置、编译、安装。

示例：make qt-reconfigure

        
make pkg-rebuild

            
            
对该源码包进行重新编译。

            
示例：
make qt-rebuild

        
make pkg-reinstall

            
            
对该源码包进行重新安装。

            
示例：
make qt-reinstall

        
make pkg-prebuilt

            
            
为该源码包生成预编译二进制压缩包。

            
示例：
make qt-prebuilt

        
make pkg-clean

            
            
删除该源码包的所有编译输出。

            
示例：
make qt-clean

        
make pkg-distclean

            
            
删除源码包的源码。

            
示例：
make qt-distclean

        
make pkg-show-build-order

            
            
显示源码包的编译依赖和顺序。

            
示例：
make qt-show-build-order
