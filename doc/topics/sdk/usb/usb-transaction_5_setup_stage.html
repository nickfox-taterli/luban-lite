Transaction 5 (Setup Stage)

30 Jan 2024

Packet 275 是核心，其中的 Setup Data 具体解析如下：

OffsetFieldSizeValueDescription
0bmRequestType10x00表示期望的 数据传输方 向为主机传 输给设备。
1bmRequest10x05代表了一个命令，根据 Table 9-4 可以得知代表 SET_ADDRESS
2wValue20x03 0x00根据 Table 9-3 可以得知，此时 代表的是： Device Address 即 ： Device Address = 0x03
4wIndex20x00 0x00根据 Table 9-3 可以得知，此时 代表的是： Zero
6wLength20x00 0x00根据 Table 9-3 可以得知，此时 代表的是： Zero
footer def
