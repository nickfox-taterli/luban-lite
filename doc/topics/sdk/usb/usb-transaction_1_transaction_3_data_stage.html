可以看到 Transaction 1 ~ Transaction 3 联合起来从 Device 传递回了 18 个字节的数据给 Host， Transaction 1 和 Transaction 2 分别传输了 8 个字节数据， Transaction 2 传递了 2 个字节数据。

6D 04 18 C0 01 43 01 02
00 01
