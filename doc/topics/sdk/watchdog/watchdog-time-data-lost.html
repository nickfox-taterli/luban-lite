wdt 设置 pretimeout 失败

15 Jul 2024

执行 wdt -s x -p y 失败，报无效的 timeout。

Cause

Linux 的 Watchdog 子系统中对 timeout 和 pretimeout 参数有限制：pretimeout 必须小于 timeout，如果不符合这个条件就会设置失败。

Remedy

过程

设置 pretimeout 小于 timeout。
