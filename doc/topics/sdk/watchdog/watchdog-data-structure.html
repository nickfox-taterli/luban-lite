数据结构设计

9 Apr 2025

Watchdog 数据类型及其结构描述如下：

  • struct aic_wdt_dev
    属于 Driver 层接口，记录 Watchdog 控制器的配置信息：
    struct aic_wdt_dev {
    rt_watchdog_t wdt;
    struct aic_wdt chan;
    s32 dbg_continue;
    u32 cur_chan;
};
  • struct aic_wdt
    属于 HAL 层接口，记录每一个 Watchdog 通道的配置信息：
    struct aic_wdt {
    u32 clr_thd;
    u32 irq_thd;
    u32 rst_thd;
};
