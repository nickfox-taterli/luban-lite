常见问题

15 Jan 2024
Read time: 1 minute(s)

使用 i2c-tools 扫描不到任何从机地址

通过下列方式，检查硬件：

  • 检查 I2C 的 SDA SCL 是否上拉到 3.3 V，I2C 空闲时是否为高电平（推荐上拉电阻 4.7 KΩ）。

  • 检查 I2C 的引脚复用，查看是否有引脚冲突。

  • 检查硬件连接是否正常，SDA SCL 是否对应连接。

触摸不正常

触摸不正常一般会有以下两种情况：

  • UI 响应位置和触摸点位置不对应

  • 触摸没有任何反应

使用以下方法排查：

  1. 打开 TP 的测试程序

    Drivers examples  --->
    [*] Enable touch panel driver test command

  2. 单跑 TP 检查坐标值，确认事件等是否正常上报。

  3. 用示波器检查 int 引脚的电平信号，点击屏幕时 int 引脚电平是否发生变化。

  4. 用逻辑分析仪检查 sda scl 的时钟和数据线的信号，通过硬件波形计算坐标值，看是否是软件处理流程不对。
