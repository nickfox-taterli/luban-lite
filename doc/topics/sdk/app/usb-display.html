USB Display 方案

16 May 2025
Read time: 1 minute(s)
如图所示，USB Display 设备通过 USB 线缆连接到 PC 设备上，在 PC 上安装完对应的驱动程序后，USB Display 设备就变成了 PC 的一个扩展显示器。

  • “PC 设备” 泛指 USB Host 端的设备，为了简化描述下文仍沿用 “PC 设备” 或 “PC”。

  • USB Device 端可以基于 Luban-Lite SDK 进行 USB Display 方案的二次开发，通过 menuconfig 菜单修改 SDK 配置参数即可完成。开发流程快速便捷，几乎不需要修改代码。
