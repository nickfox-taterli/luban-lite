在 Windows 中校准触摸屏

15 Jan 2024

在 Windows 环境下，默认触摸屏会控制主屏。为了使触摸屏能够控制 USB 副屏，需要设置相关参数。

详细设置流程如下所示：

  1. 在 Windows 开始菜单中找到控制面板选项。

  2. 在控制面板界面，打开硬件和声音 > 平板电脑设置选项。

  3. 在弹出的对话框中，点击设置。
    点击设置后，系统会弹出下列提示：

  4. 按 Enter 键让上述字体显示到副屏上，并点击副屏的触摸屏直到界面消失。

    此时可直接在副屏上使用触摸屏操控。
