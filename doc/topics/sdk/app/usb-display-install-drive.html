安装驱动

9 Apr 2025
Read time: 2 minute(s)

如果 PC 设备之前未安装对应驱动，当 USB Display 设备通过 USB 接入 PC 设备后，系统会多出一个 U 盘设备，其中包含需要的驱动安装包文件。

针对不同的操作系统，PC 设备驱动的安装方法也不仅下同。

在 Windows 系统中安装驱动

在 Windows 系统中，当 USB Display 设备通过 USB 接入 PC 设备后，在 U 盘中会看到一个 .exe 后缀的 USB Display 驱动安装文件，如下图所示：

双击 USB Display 驱动安装文件，并按照对话框指示说明，完成安装即可。

在 Linux 系统中安装驱动

  1. 确保使用 Xorg 图形系统。

    Linux 版本的 USB Display 当前仅支持 Xorg 图形系统。如果当前系统使用了 Xwayland，必须先切换到 Xorg 图形系统。不同操作系统中的切换流程不尽相同，详情可查看切换到 Xorg 图形系统。

  2. 执行安装脚本。

    在 Linux 系统中（含树莓派环境）中，当 USB Display 设备通过 USB 接入 PC 设备后，U 盘中提供了一键安装的 .sh 脚本文件：

    • install_for_linux.sh：安装 USB Display 的脚本

    • uninstall_for_linux.sh：卸载 USB Display 的脚本

    在 Linux 系统中安装软件和运行上述脚本均需要 root 用户权限，建议通过终端命令行来安装。详细安装流程如下：

    1. 进入 U 盘文件夹后，选中 Linux 文件夹并点击鼠标右键。

    2. 在弹出的菜单栏中，选择 Open in Terminal(在终端中打开)。

      此时，终端 界面会弹出。

    3. 在终端命令行中，执行下列命令安装脚本：
      sudo bash install_for_linux

      安装成功后，系统会提示 AicUSBDisplay installed successfully。

    4. 重新插拔一次 USB。

      正常情况下，USB 扩展屏可正常显示。

    5. 如需卸载，重复上述操作打开命令行终端，并执行下列卸载脚本：
      sudo bash uninstall_for_linux

切换到 Xorg 图形系统

Linux 版本的 USB Display 当前仅支持 Xorg 图形系统。如果当前系统使用了 Xwayland，必须先切换到 Xorg 图形系统。详细切换流程如下所示：

  • 在树莓派中切换 Xorg 图形系统：
    1. 在终端中执行下列命令，通过 raspi-config 工具来修改图形库的类型：
      sudo raspi-config
    2. 在下列配置选项中选择 X11 后，按 ESC 按键退出配置界面。
      -> 6 Advanced Options
    -> A6 X11/Xwayland

    3. 重启系统

  • 在 Ubuntu 中切换 Xorg 图形系统：
    1. 在在终端中执行下列命令：
      sudo vim /etc/gdm3/custom.conf

    2. 按照下列操作修改 GDM 配置文件，保存配置并退出。

      在文件编辑界面，找到 WaylandEnable=false 行，删除注释符号 # 后打开该行配置。

    3. 重启系统。
