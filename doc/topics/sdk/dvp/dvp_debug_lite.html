16 May 2025

驱动加载状态

DVP 的设备初始化入口，是放在 MPP VIN 的初始化中调用完成的。

在 RT-Thread 中，DVP 暂时没有注册为一个设备，只是提供了一些 API 接口，所以从命令 list_device 中是看不到 DVP 设备，但是可以看到 Camera 设备：
list_device
device           type         ref count
-------- -------------------- ----------
...
camera   Camera Device        0
...

调试开关
