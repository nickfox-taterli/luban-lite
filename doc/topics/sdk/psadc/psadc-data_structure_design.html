数据结构设计

5 Jul 2024

struct aic_psadc_dev

属于 Driver 层内部使用的数据结构，管理 PSADC 控制器的设备资源：
struct aic_psadc_dev {
    struct rt_adc_device *dev;
    struct aic_psadc_ch *chan;
};

struct aic_psadc_ch

属于 HAL 层接口，记录各个通道的配置信息：
struct aic_psadc_ch {
    u8 id;
    u8 available;
    enum aic_psadc_mode mode;
    u8 fifo_depth;

    aicos_sem_t complete;
};
