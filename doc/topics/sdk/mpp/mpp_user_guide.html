MPP 使用指南

2 Dec 2024
Read time: 1 minute(s)

Media Process Platform (MPP) 是 ArtInChip 自主研发的通用多媒体处理软件平台，适用于 ArtInChip 芯片系列。支持在 RTOS 平台上运行， 屏蔽了 ArtInChip 不同芯片平台多种多媒体硬件模块（VE、GE 等）版本的差异，为使用者提供简单易用的多媒体处理 API，支持多种多媒体解决方案。

目前支持硬件模块包括：

  • VE：视频、图片编解码功能

  • GE：2D 图形加速

表 1. 术语定义
术语定义注释说明
