配置横屏竖用和竖屏横用功能

9 Apr 2025
Read time: 1 minute(s)
在实际使用过程中，如需使用横屏竖用和竖屏横用功能，可根据下列流程进行配置：
  1. 在 SDK 根目录，执行下列命令，进入 menuconfig 的功能配置界面：
    scons menuconfig

  2. 配置显示的旋转角度：

    > Board options  --->
   [*] Using Display Engine (DE)
   Display Parameter  --->
       framebuffer rotation degree (0)  --->
           (X) 0
           ( ) 90
           ( ) 180
           ( ) 270
  3. 配置 TP 的旋转角度和镜像。
    > Drivers options  --->
   Peripheral  --->
       [*] Touch Panel Support  --->
           [ ]   Touch filp x coordinate
           [ ]   Touch filp y coordinate
           [ ]   Touch 90 degrees rotation
           [ ]   Touch 270 degrees rotation
